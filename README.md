#  Windows User Management & Software Installation 

##  Project Overview

This project demonstrates basic Windows system administration tasks using PowerShell.  
The following tasks were completed:

- Creating and managing local users
- Creating and managing local groups
- Assigning folder permissions
- Installing software using Chocolatey
- Troubleshooting installation issues

All tasks were performed using **PowerShell (Run as Administrator)**.
